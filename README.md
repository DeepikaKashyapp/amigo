# 🌿 Chaitanya - AI Mental Health Support System

Chaitanya is a localized, AI-powered mental health support platform designed specifically for students. Built for the Smart India Hackathon (SIH), this platform focuses on accessibility for rural and semi-urban users by prioritizing culturally relevant interactions and anonymous support over formal, intimidating clinical environments.

## ✨ Key Features
* **Conversational AI Chatbot:** Intelligent, supportive interactions powered by Google Gemini.
* **Localized Branding:** Strategically designed with culturally relevant Hindi naming conventions to maximize end-user adoption and comfort.
* **Anonymous Counseling:** Secure, judgment-free space for students to express their emotions.
* **Mood Tracking & Wellness:** Continuous assessment and personalized mental health resource recommendations.
* **Gamified Engagement:** Interactive features designed to improve user retention and emotional well-being.

## 🛠️ Tech Stack
* **Frontend:** React.js, JavaScript, Tailwind CSS
* **Backend:** Node.js
* **AI & APIs:** Google Gemini AI (via Google AI Studio)

---

## 🚀 Run Locally

**Prerequisites:** Node.js installed on your system.

### 1. Install dependencies:
```bash
npm install
```
###2. Environment Setup:
```
Create a .env.local file in the root directory and set your Gemini API key:
Code snippet
GEMINI_API_KEY=your_api_key_here
```
###3. Run the app:
```Bash
npm run dev
```
